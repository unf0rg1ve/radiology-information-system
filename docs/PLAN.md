## Независимые задачи

### 1. Импорт тарификатора из Excel/CSV (M6 — F6.1)

**Файлы:** `backend/scripts/seed_refs.py`, возможно новый `backend/app/api/imports.py`

**Что сделать:**
- Реализовать эндпоинт `POST /api/refs/services/import` с загрузкой Excel/CSV
- Валидация обязательных колонок (code_gombp, name_ru, modality, tariff_gombp, duration_min)
- Версионирование: при импорте старые записи `is_active=False`, новые `version+1`
- Прогресс и сводка ошибок в ответе

**Затрагивает:** только backend

### 2. Учёт недоступности аппарата (M3 — F3.3)

**Файлы:** `backend/app/api/schedule.py`, `backend/app/models/device.py` (возможно новая модель или поле)

**Что сделать:**
- Добавить в модель Device или отдельную таблицу `device_unavailability` для окон ТО/поломки
- При генерации слотов (`GET /api/schedule/slots`) исключать эти окна
